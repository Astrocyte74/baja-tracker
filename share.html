Open this on the traveler's iPhone, save the gist details once, then tap one button to update the Baja map. No iOS Shortcut setup required.

Phone Setup

The token is stored only in this browser on this phone using local storage. Do not use a personal all-access token here; create a dedicated token with only gist scope.
Enter the gist ID and token once, then use the main button whenever you want to update the map.
Tip: Safari -> Share -> Add to Home Screen makes this a one-tap updater.
