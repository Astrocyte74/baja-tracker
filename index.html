ℹ️ How This Works

Your friend runs an iOS Shortcut that sends their location to this map. The shortcut uses GitHub Gist to store the location data.

📱 iOS Shortcut Setup

Download the shortcut and configure it with your GitHub Gist ID and token.

Step 1: Create a GitHub Gist

Go to gist.github.com and create a new gist:

{
  "name": "Traveler",
  "lat": 25.0,
  "lng": -111.0,
  "timestamp": "2024-01-01T00:00:00Z"
}

Copy the Gist ID from the URL (the long alphanumeric string after gist.github.com/)

Step 2: Create a GitHub Personal Access Token

Go to GitHub Token Settings

  • Name it "Baja Tracker"
  • Select only gist scope
  • Copy the token

Step 3: Add Shortcut Configuration

Edit the shortcut and replace these values:

GIST_ID: "your-gist-id-here"
GITHUB_TOKEN: "your-token-here"
